# Age Group Prediction

## Overview
Built a machine learning model to predict user age groups using demographic and behavioral data.

# Tools
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

# Workflow
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation

# Models Used
- Logistic Regression
- Decision Tree
- Random Forest

# Result
Achieved 84.53% validation accuracy using a Random Forest classifier.
